# Digital-clock

A simple and responsive digital clock built using HTML, CSS, and JavaScript. This project displays the current time in real-time and updates every second.
  
---
# Live Demo

https://collins-cloud.github.io/Digital-clock-/
---
# 📌 Features
- Displays current hours, minutes, and seconds
- Updates automatically every second
- Uses JavaScript setInterval() for real-time updates
- Clean and simple UI
- Beginner-friendly project
- Fully responsive design

---

# Built with
- HTML5
- CSS3
- JavaScript (Vanilla JS)
---

# 💡 What I Learned
- How to use the JavaScript Date() object
- How to use setInterval() to update elements in real time
- How to manipulate the DOM
- How to structure a simple project
